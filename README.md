# Linux-подобная Операционная Система на Python

Добро пожаловать в нашу учебную Linux-подобную операционную систему, реализованную на Python. Это учебный проект, который демонстрирует основные принципы работы операционных систем с командным интерфейсом.

## Возможности системы

Наша система предоставляет следующие функции:
- Навигация по файловой системе
- Создание и удаление файлов и директорий
- Просмотр содержимого файлов
- Информация о системных ресурсах
- Основные команды, похожие на Linux

## Установка и запуск

Для работы системы вам понадобится:
- Python 3.6 или выше
- Библиотека `psutil`

Установка библиотеки:
```bash
pip install psutil
```

Запуск консольной версии системы:
```bash
python linux_console_os.py
```

## Доступные команды

- `ls [path]` - список файлов и директорий
- `pwd` - текущая директория
- `cd <path>` - сменить директорию
- `echo <text>` - вывести текст
- `mkdir <name>` - создать директорию
- `touch <file>` - создать файл
- `cat <file>` - вывести содержимое файла
- `rm <file/dir>` - удалить файл или директорию
- `rmdir <name>` - удалить пустую директорию
- `ps` - список процессов
- `df` - информация о дисковом пространстве
- `free` - информация об использовании памяти
- `date` - текущая дата и время
- `help` - справка по командам
- `clear` - очистить экран
- `exit` - выйти из системы

## Структура проекта

- `linux_console_os.py` - основной файл с консольной версией системы
- `linux_os_simulator.py` - файл с GUI версией (требует установленный Tkinter)

## Особенности реализации

Система реализована с использованием стандартных библиотек Python, а также библиотеки `psutil` для получения информации о системных ресурсах. В консольной версии реализованы все основные функции, аналогичные тем, что есть в настоящих Linux-системах.

Обратите внимание, что это учебный проект и не предназначен для использования в продакшене. Все операции выполняются в пределах вашей текущей файловой системы.
